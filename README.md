# MavericsTest
--------------------------

Minimum Requirements
====================
- Android studio 3.2 beta
- Android SDK 19 (KitKat)
- Java 8 (Update higher than 80 because of a known issue with Android Studio on with older versions)

Architecture
============
### Jetpack architecture components
- DataBinding
- Room
- ViewModel
- Lifecycles

### Networking
- OkHttp
- Dagger2
- Retrofit

### Image library
- Glide
